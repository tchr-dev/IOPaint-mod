## Best Practices for AI Agents

1. **Always check docs first** - Before asking questions, consult relevant documentation
2. **Use the planning system** - Create planning tasks for significant work
3. **Update documentation** - If you discover something not documented, add it
4. **Follow naming conventions** - Use lowercase kebab-case for new files
5. **Link related documents** - Cross-reference relevant planning and architecture docs
