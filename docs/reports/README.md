# Reports

Audit reports, analysis results, and metrics for the IOPaint project.

## Contents

- [npm-audit/](npm-audit/) - NPM security audit reports

## Contributing Reports

When adding new reports:

1. Create a date-prefixed file in the appropriate subdirectory
2. Update the relevant index (e.g., npm-audit/README.md)
3. Include a summary at the top of the report
4. Add any remediation steps or notes

## Report Format

For date-prefixed reports: `YYYY-MM-DD-descriptive-title.md`

Include:
- Report date
- Summary of findings
- Detailed results
- Remediation steps (if applicable)
- Links to related resources
