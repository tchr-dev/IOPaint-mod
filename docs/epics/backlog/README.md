## Adding to Backlog

When adding a new epic to the backlog:

1. Copy `epics/template.md` to this directory
2. Fill in Vision, Scope, and Success Criteria
3. Identify initial sub-tasks (to be expanded in planning/)
4. Set priority and estimated duration
5. Add to this README's priority queue table
